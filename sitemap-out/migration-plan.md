## 4) ลิงก์เสีย — อันไหนข้ามได้ / อันไหนต้องตรวจมือ

- **หายจริง: 0 รายการ** → ไม่มีลิงก์ตายที่ต้องกังวล 🎉
- **ชื่อไฟล์ไทยเพี้ยน: 827 รายการ** → **ข้ามได้ในแง่ "หน้าหาย"** เพราะหน้าปลายทาง
  มีอยู่จริงบนดิสก์ แค่ชื่อไฟล์ภาษาไทยถูก HTTrack เก็บเสีย (tis-620 + U+FFFD)
  ทำให้ match อัตโนมัติไม่ได้

  **สิ่งที่ต้องทำตอนย้าย:** หน้าไทยพวกนี้ (วิธีชำระเงิน, ติดต่อเรา, เงื่อนไข ฯลฯ)
  ให้ **เปิดดูเนื้อหาด้วยตา → ตั้ง slug/URL ใหม่เป็นภาษาอังกฤษหรือ utf-8**
  เช่น `payment`, `contact`, `how-to-order` แล้วทำ redirect map เอง
  (มีอยู่แค่ไม่กี่หน้า ทำมือได้)
